eShop Proxy

    Forward requests with 'kubectl port-forward deployment/apache 8080:80'

    Catalog
    ProxyPass        /categories http://category-service:8000/v1/categories
    ProxyPassReverse /categories http://category-service:8000/v1/categories
    Example requests:
        - POST  http://localhost:8080/categories
        - GET   http://localhost:8080/categories/1

    Product
    ProxyPass        /products http://product-service:8001/v1/products
    ProxyPassReverse /products http://product-service:8001/v1/products
    Example requests:
        - POST  http://localhost:8080/products
        - GET   http://localhost:8080/products/1
